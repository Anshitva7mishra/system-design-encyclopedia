## Who Am I

I am **Anshitva Mishra**, a final-year B.Tech CSE student learning System Design from scratch — in public.

This repository is my daily study notebook. No polished summaries written after the fact. No copied resources. Just raw, honest notes and Java implementations from each day of learning.

If you are also learning System Design, follow along. Everything here is free.

---

## How This Repository Works

Every day I study, I create a new folder.

```
DayX/
├── notes.md        ← what I learned, in my own words
└── code/           ← Java implementations from that day
```

That is it. Simple, honest, consistent.
